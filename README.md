## openafrR — A Grammar of African Data Science (GAfDS)

openafrR is a metapackage ecosystem designed to unify African statistical, economic, financial, logistics, and governance datasets into a reproducible data science framework.

The project aims to provide country-level packages for all 54 African countries.

Current ecosystem includes:
- openngaR — Nigeria
- openzafR — South Africa

Long-term goals:
- Reproducible African macroeconomic data
- Government statistical interoperability
- Tax and compliance analytics
- Public-sector data engineering
- Open African data infrastructure
